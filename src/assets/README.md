# Assets

Organized by purpose. Use **kebab-case** file names.

| Folder | Contents |
|--------|----------|
| `charts/` | Figma SVG chart sources (bundled at build via `npm run generate:charts`) |
| `facial-analysis/` | Dashboard portrait images (desktop + mobile) |
| `hero/` | Hero-only assets (e.g. motion line reference SVG) |
| `photos/` | Before/after comparison photos |
| `ambient/` | Large blurred background images for sections |
| `icons/` | UI icons |
| `philosophy/` | Philosophy section card illustrations |
| `video/` | Source video (`landing-video.mp4` → `public/videos/` on dev/build) |
